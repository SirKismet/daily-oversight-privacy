How We Can Help

Contact us for help using Daily Oversight, to report a problem, submit feedback, or request a feature.

When Reporting a Problem

To help us understand and resolve an issue, please include:

  • A brief description of what happened.
  • The steps you took before the issue occurred.
  • Your iPhone model.
  • Your iOS version.
  • The version of Daily Oversight you are using.

Basic Troubleshooting

  • Make sure you are using the latest version of Daily Oversight.
  • Make sure your iPhone is using the latest available version of iOS.
  • Close and reopen the app.
  • Restart your iPhone if the issue continues.
